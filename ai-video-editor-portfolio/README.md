# AI Video Editor Portfolio (React + Tailwind + Vite)

## Quick start
```bash
npm install
npm run dev
```
Then open the shown local URL.

## Build
```bash
npm run build
npm run preview
```

## Replace your photo
Edit `src/components/Hero.jsx` and replace the placeholder image URL.
